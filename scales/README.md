# Escalas Clínicas Geradas

Esta pasta contém os arquivos Python (.py) das escalas clínicas extraídas e geradas pela aplicação Clinical Scales Extractor.

## Sobre

Cada arquivo nesta pasta representa uma escala clínica que foi processada através do sistema, contendo:
- Nome e descrição da escala
- Referência bibliográfica
- Variáveis necessárias para o cálculo
- Método `calculate()` implementado em Python

## Uso

Os arquivos Python gerados podem ser importados e utilizados em outros projetos para realizar cálculos de escalas clínicas de forma programática.

## Aviso

⚠️ **IMPORTANTE**: Sempre verifique os resultados com calculadoras oficiais (como MDCalc) antes de usar em decisões clínicas reais.
